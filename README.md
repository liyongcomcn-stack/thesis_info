# thesis_info

此仓库用于存放与论文相关的信息与资料索引，包含论文题目、作者信息、参考文献、数据集说明、实验记录、以及与论文写作和排版相关的模板与脚本。仓库旨在作为论文研究的辅助工具，方便团队成员查阅与协作。

## 目录结构（示例）
- /papers         - 已发表或参考的论文 PDF
- /data           - 实验数据与说明
- /notebooks      - 数据分析与可视化的 Jupyter 笔记本
- /scripts        - 实验复现脚本与工具
- /templates      - 论文模板（LaTeX/Markdown）
- README.md       - 项目说明（本文件）

## 主要内容
1. 论文列表与摘要：记录研究背景、动机、关键结论和待完成事项。
2. 数据与复现实验：提供数据获取说明、预处理步骤和重现实验的脚本。
3. 模板与写作规范：包含论文排版模板、参考文献样式及提交清单。

## 使用方法
1. 克隆仓库：
   git clone https://github.com/liyongcomcn-stack/thesis_info.git
2. 浏览各目录，按 README 或目录下说明执行对应脚本。
3. 若需复现实验，请先查看 /data 与 /scripts 下的说明文件，确保安装所需依赖。
